Unity est le moteur le plus utilisé avec une énorme communauté et des milliers de tutoriels gratuits. C# a une syntaxe accessible quand on connaît déjà le C. ​ Python avec Pygame - Parfait pour créer rapidement des prototypes de jeux et comprendre les mécaniques de base. Python est facile à apprendre et très polyvalent pour d'autres projets. ​ C++ avec Unreal Engine - Vous avez déjà des bases en C, donc C++ sera une progression naturelle.
